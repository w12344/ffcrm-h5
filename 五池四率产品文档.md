三大工作台 — 气泡池产品说明文档
一、线索池
1. 数据来源： 系统分配或自主拓展的线索（学生），即我负责的线索；家庭群线索只展示学生气泡，避免视觉冗余；

2. 数据权限： 仅查看我负责的线索；

3. 数据展示方式： 气泡图 / 表格；

4. 气泡图展示说明：

气泡大小： 根据线索价值等级（高价值 / 普通 / 垃圾）输出预定义气泡尺寸；高价值线索气泡最大，垃圾线索气泡最小且固定尺寸；

气泡颜色： 根据线索进入系统后的经过时间，输出预定义气泡颜色：

最新线索（0–2 小时）：玫红→紫色渐变；
活跃线索（2–24 小时）：橙色→粉色渐变；
沉睡线索（超过 24 小时）：灰色磨砂渐变；
气泡位置： 分为水面区域和池底区域；

水面区域： 展示最新线索和活跃线索（进入系统 24 小时以内）；根据线索进入时间倒序，气泡从右向左依次排列，气泡相切排布；第一行放满后自动换到下一行，逐行从上向下堆叠；
池底区域： 展示沉睡线索（进入系统超过 24 小时未跟进的线索）；根据进入时间倒序，气泡从右向左依次排列；
气泡警示标识（存在风险/问题）： 线索存在撞单风险，或者线索被标记为垃圾线索；

气泡闪烁（存在机会）： 线索为高价值线索；

展示全量我的线索数据；

5. 坐标轴说明：

X 轴：线索进入时间排序（从右到左，最新的在右边）；
Y 轴：分为水面区域（上方）和池底区域（下方），由系统自动排布；
二、客户池 & 升级池
1. 数据来源： 已匹配销售人员的客户（学生），即我负责的客户（学生）；家庭群客户只展示学生气泡，避免视觉冗余；

2. 数据权限： 仅查看我负责的客户；

3. 数据展示方式： 气泡图 / 表格；

4. 气泡图展示说明：

气泡大小： 根据客户质量分（肥瘦分 ∈ [0, 10]），输出预定义气泡尺寸；肥瘦分越高气泡越大，代表客户购买力、匹配度等基础盘越好；特殊情况：肥瘦分为 0 或者没打分，固定气泡尺寸（最小）；

气泡颜色： 根据客户综合评级（A / B / C / D / X），输出预定义气泡颜色：

A 级：玫红→紫色渐变（最优质客户）；
B 级：橙色→粉色渐变；
C 级：绿色→蓝色渐变；
D 级：青色→紫色渐变；
X 级：灰色磨砂渐变（盲盒客户，尚未评级）；
气泡位置： 分为水面区域和池底区域；

水面区域： 展示活跃客户（未放弃且 1 周内有沟通的客户）；根据最近沟通时间倒序，气泡从右向左依次排列，气泡相切排布；第一行放满后自动换到下一行，逐行从上向下堆叠；
池底区域： 展示顾问主动放弃的客户，或者超过 1 周未沟通的客户；根据放弃时间或最近降级时间倒序，气泡从右向左依次排列；多余客户自动流向左侧（支持横向滚动）；
气泡警示标识（存在风险/问题）： 客户存在待解决异议，或者 AI 识别的相关风险；

气泡闪烁（存在机会）： 客户成熟度 90 分以上但未成交，或者 AI 识别的相关机会；

展示全量我的客户（学生）数据；

5. 坐标轴说明：

X 轴：最近沟通时间排序（从右到左，最近沟通的在右边）；
Y 轴：分为水面区域（上方）和池底区域（下方），由系统自动排布；
三、成交池 & 入学池
1. 数据来源： 已交费（定金/全款）或已正式入学的客户（学生），即我负责的已成交客户；家庭群客户只展示学生气泡，避免视觉冗余；

2. 数据权限： 仅查看我负责的已成交客户；

3. 数据展示方式： 气泡图 / 表格；

4. 气泡图展示说明：

气泡大小： 根据客单价/产品持有量，输出预定义气泡尺寸；购买了全套产品或全款交费的客户，气泡尺寸最大；仅交纳少量定金的客户，气泡为标准尺寸；

气泡颜色： 根据交付健康度状态，输出预定义气泡颜色：

已入学（健康交付）：绿色→蓝色渐变；
待开班（预收阶段）：橙色→粉色渐变；
已退费：灰色渐变；
气泡位置： 分为水面区域（入学池 / 安全交付区）和池底区域（成交池 / 预收与风险区）；

水面区域（入学池）： 展示正式入学并与教务系统打通的学生；根据入学时间（优先）或付款时间倒序，气泡从右向左依次排列，气泡相切排布；第一行放满后自动换到下一行，逐行从上向下堆叠；
池底区域（成交池）： 展示交了钱（定金/全款）但还没正式入学的客户，以及发生退款的客户；根据付款时间倒序，气泡从右向左依次排列；退费客户固定沉底显示；
气泡警示标识（存在风险/问题）： 存在付款撞单风险（同一客户被多个顾问收款）；

气泡闪烁（存在机会）： 存在复购/续费机会（如即将到期的课程包、可升级的产品等）；

展示全量我的已成交客户（学生）数据；

5. 坐标轴说明：

X 轴：入学时间/付款时间排序（从右到左，最近的在右边）；
Y 轴：分为水面区域（入学池，上方）和池底区域（成交池，下方），由系统自动排布；
四、三池通用视觉规范
1. 气泡排列规则（三池统一）：

所有气泡从右向左依次排列，越新/越活跃的排在越右边（最显眼位置）；
每行气泡水平方向严格相切排列（无额外间距），第一行放满后自动换行；
行与行之间纵向相切堆叠（类似俄罗斯方块从上往下紧密排列）；
2. 气泡动效与徽章（三池统一）：

每个气泡有独立的轻微漂浮动画（随机方向、随机速度），模拟水中浮动效果；
警示标识为气泡中心的黑色感叹号「!」，带白色发光效果；
机会闪烁为气泡整体透明度脉冲动画（明暗交替），提示顾问关注；
气泡底部可展示产品持有徽章（如 A、B、C、D 产品标签），采用毛玻璃圆角矩形样式；
3. 头部信息栏（三池统一）：

左侧：返回按钮 → 池子名称 → 彩色占比条（各分类的比例可视化，鼠标悬停显示详情） → 客户/学生总数；
右侧：颜色图例（仅展示颜色圆点+状态名称） → 列表查看按钮（点击切换为表格视图）；
4. 背景与网格（三池统一）：

背景透明，叠加轻量科技感粒子动画；
Y 轴左侧与 X 轴底部各有浅色网格刻度线，辅助感知空间位置；
水面区域和池底区域之间有自然的视觉间隔（约 5%–15% 的留白区域）。
