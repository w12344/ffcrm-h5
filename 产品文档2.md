一、 核心诉求与设计理念
- 一秒钟看清公司盈亏与风险（钱袋子与护城河）。
- 两秒钟看透谁在赚钱谁在拖后腿（团队战力与质量）。
- 三秒钟预判下周的业绩稳不稳（未来粮草与转化）。
- 最后能直接给出管理动作（挥鞭子与发奖励）。

二、 核心功能模块详细说明
模块一：全局大盘
采用“1主指标+1趋势标签+2核心子指标”的极简结构，直击业务命脉。
💳 卡片 1：全局营收与基本面（看大盘底座）
- 主指标：总营收完成额（调取大盘真实收款合计，如 6416万）。
  - UI 辅助标签：↓ 同比去年 -463.6万（红绿箭头直观展示年度增长趋势）。
- 核心子指标 1：目标营收缺口（如 677万）。用总目标减去已完成额，直接向全员暴露“今年还差多少钱达标”，作为冲刺 KPI 抓手。
- 核心子指标 2：净招生人头数（如 936人）。严格扣除退费后的实际招生源数，直观反映后端教务的排课与交付压力。
⚠️ 卡片 2：风险与欠款池（看利润流失与催收）
- 主指标：总待收款 / 欠款池（如 1250万）。落实“成交池显示欠费标识”规则，随时展示已定权但未收齐的“准现金”。
  - UI 辅助标签：涉及 48 个欠费单子，方便老板下达催收指令。
- 核心子指标 1：大盘总退费金额（如 404.1万，附带趋势标签 ↓ 同比减少 147.9万）。直击利润流失黑洞，时刻警醒全员抓风控。
- 核心子指标 2：累计退费人数（如 259人）。落实“退费率由顾问控制”的考核原则。
🎯 卡片 3：未来产出与转化（看下周粮草）
- 主指标：当下激活客户数（如 559个）。落实核心考核定调：“不用管激活率，只看当下激活数，这决定了下周能否产出”。
  - UI 辅助标签：✨ 下周业绩预测。
- 核心子指标 1：上门成交率（如 61%）。衡量团队将优质线索转化为真金白银的“临门一脚”逼单能力。
- 核心子指标 2：综合留存率（如 67%）。闭环教育生意的长线未来，直观反映教务交付质量与老生续费能力。

三、 团队能效雷达（透视员工的“绩效”与“工作量”）
用科技感的可视化图形代替枯燥表格，实时透视每个销售顾问的战力与状态。
[图片]

1. 中心大盘数据球
- 展示内容：公司总目标达成率（如 90.4%）及总目标金额（如 目标 7093万），居中高亮展示大盘进度。
2. 顾问卫星气泡与智能分级
- 气泡颜色（状态预警）：系统根据《PRD》公式（实际销售额 / 目标进度）自动打分分级。
  - A级 (≥110%)：紫色/粉色（优秀标杆，如张莹）。
  - B级/C级 (90%-109%)：绿色/橙色（正常推进）。
  - D级预警 (<90%)：红色高危刺眼显示（如刘祥宇、陈孟丽），老板只需满屏找红球即可精准约谈落后员工。
3. 🔍 鼠标 Hover 气泡透视（老板专属 X 光）
当鼠标悬停在特定员工（如张莹）的气泡上时，直接弹出以下 5 项能看透其虚实的底层数据：
1. 业绩贡献：营收完成额 & 净人头数。
2. 风控黑洞：个人退费金额 / 退费人数（抓签单水分大的人）。
3. 准现金流：个人待收款（下达一对一催收指令）。
4. 下周粮草：当下激活客户数（预判他下周有没有单出）。
5. 转化能力：上门量 & 上门成交率（看线索是否被浪费）。

四、 AI 智能管家与管理抓手（鞭策面板）
老板不需要自己去数据里找问题，通过AI分析数据把“标杆”和“问题”直接推到老板面前。
将原有的“顾问助手”全面重构为老板的**【每日高管日报与督办指令面板】**，实现“发现问题 -> 一键挥鞭”的极简管理闭环。
1. 单卡片推送逻辑
- 系统每日自动生成一张极度克制的汇总卡片推送到面板，包含“标杆榜样”与“红线预警”两大部分。
2. 🏆 业绩标杆 TOP 3（立标杆）
- 调取营收数据，列出当前表现最好的前 3 名（如：张莹、吴亚丽、金雷拉）及达成率，提供群内表扬建议。
3. ⚠️ 红线违规预警清单（抓问题）
系统基于底层底线规则，自动揪出以下四类异常员工：
- 业绩严重滞后（D级）：当前进度低于 90% 的员工（如刘祥宇进度仅44%）。
- 资源浪费预警：拿着线索不干活，建联率 < 60% 的员工（如廖正凯）。
- 下周断粮预警：“当下激活客户数”极低的员工（如龙青仅9个）。
- 沉睡线索流失预警：手里有高价值线索但超过 7 天未跟进的员工（如赵菲菲）。
4. 一键管理抓手（批处理动作）
在卡片最底部提供快捷操作按钮，老板点击后系统将自动在后台执行指令：
- [ 群发战报与表扬]：一键将 TOP 3 成绩发送至全员企微群。
- [ 预警员工强提醒]：老板点击后，系统会向上述所有违规/落后员工的屏幕强制弹出带有震动效果的“大弹窗”，且必须点击确认才能继续工作
